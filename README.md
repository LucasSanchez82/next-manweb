
# Manweb

Manweb is an application for memorizing your reading mangas


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

To deploy this with docker to http://localhost:3000

```bash
  docker-compose up
```
